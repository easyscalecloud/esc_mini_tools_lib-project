Install
------------------------------------------------------------------------------

``esc_mini_tools_lib`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install esc-mini-tools-lib

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade esc-mini-tools-lib
